# HTML
stands for Hyper Text Markup Language. It puts the content, text on the website and sets up the layout.

## Level 1 (HTML Basics)

1) File Extension :- .html or .htm(not used nowadays).

2) index.html :- default name of website homepage

3) tag :- is a basic unit of html and it is used to create html elements.

4) comments :- used to add notes in code for understanding. <!--This is a comment-->

5) Case sensitivity :- it is case-insensitive for tags and attribute names.

## Level 2 (Must-Use HTML Tags)

1) Html attributes :- Provides addtional information about elements like height, weight, src, href, etc. {name=value}.

2) id property :- Each id should be unique within a page. Allows for direct links to sections using the #id syntax in URLs.

3) HTML Tags :- h1 to h6, p, br, hr, image, video, anchor, bold/italic/underline/strikethrough, pre, big/small, superscript/subscript

4) Character Entity Reference :- Used to display special or reserved characters, like this &copy;.

## Level 3 (Browser Tools)

1) Browser Tools :- View Page Source displays raw HTML and CSS. Inspect Element allows real-time editing of HTML/CSS.
HTML without CSS is just skeleton without muscles.

2) Responsive Design :- Enhances user experience on different screen sizes like mobile and desktop.

3) Live Edit Code :- We can change html, css and javascript at our machine of an website but it is temporary and happening only at our machine in that particular session.

4) Validating Webpages :- validator.w3.org can be used to check the standard of the webpage code.

## Level 4 (HTML and Project Structure)

1) Semantic Tag / Non-Semantic Tag :- Semantic tags are useful for SEO and accessibility. Non-semantic are used for layout. Like header, footer, article, section, nav are semantic tags and div, span, p are non-sematic tags.

2) Body Tags :- header, main, footer, article, section , aside tags.

3) Recommended Folder Structure for Vanilla JS(HTML, CSS, JS) Project :- ROOT directory -> .html files, css folder, js folder, images, assets, pages for multi-page website

4) More Tags :- Naviagtion tags(nav), block vs inline elements, div is a block element acts as a container for other elements. span is a inline element similiar as div.


## Level 5 (List, Tables and Forms)

1) List tags :- Ordered list(ol -> 1,A,a,I,i), Unordered list(ul -> bulleted list).

2) Table tag :- tr,td,th and caption tag for title and description must be just after table tag, colspan="3".

3) Form tag :- input tag, action attribute submits the data to the server or mentioned url. name and value attributes submitted to server, label tag connects with input tag, Input type -> Date, File, color, range, button, submit, radio, checkbox and select tag -> option tag, textArea tag

4) iFrame tag :- Allows you to embed another webpage or multimedia content within a webpage. <iframe></iframe>
